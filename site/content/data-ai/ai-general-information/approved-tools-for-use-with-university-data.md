When a user is logged in with Syracuse University credentials (NetID), Syracuse University supports the use of the following University approved tools for university data:

- [Anthropic Claude Enterprise](https://claude.ai/) - Request Access at [getclaude.syr.edu](https://getclaude.syr.edu)
- [mentorAI for Syracuse University](https://mentor.ai.syr.edu/)
- [Microsoft Copilot Work or Web](https://www.office.com/?auth=2)
- [Google Gemini](https://gemini.google.com/app) - login with NetId@g.syr.edu
- [Blackboards AI](https://blackboard.syracuse.edu/)
- [OpenAI ChatGPT Teams](https://chatgpt.com/) - Request access via IT Support
- [Adobe Firefly](https://firefly.adobe.com/)
- Gradescope
