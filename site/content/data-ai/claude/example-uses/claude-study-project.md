Create a *Study Project* inside Claude to master subjects. Upload lecture slides, PDFs, or notes and ask Claude to explain, quiz , summarize chapters or generate flashcards.

How to use it:

1. Click on Learn or Study project from the home screen.
2. Upload class materials via the project setup.
3. Ask Claude: *Summarize this PDF into flashcards* or *Explain the key concepts from this slide.*
