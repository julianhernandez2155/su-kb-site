Claude is the primary AI assistant supported by Syracuse University's ITS Data & AI department. SU runs an enterprise plan with training opt-out, custom retention, and premium access for approved IT staff and programs.

Data classification

University data with Claude

Use the syr.edu-signed-in Claude account for any work involving university data. The personal Claude account is not approved for FERPA-protected information.

The Approved Tools list is the canonical source for which AI tools are approved for which data classifications. Claude is approved for public and internal data; consult IT before using with confidential or FERPA-protected material.

Retention and training

  • Chat retention: 2 years by default on Claude.ai with SU's enterprise plan
  • Training opt-out: SU's enterprise plan opts all university chats out of model training
  • Travel: Claude can be used while traveling abroad subject to SU's international data-handling policies

Premium seats

Premium Claude seats (with API access and Claude Code) cost approximately $2,400/year per seat. Available to IT staff and specific approved programs.

Requesting a seat

Request via the Claude Code Setup flow. Approval typically takes 3–5 business days.

Setup & access

New users start at getclaude.syr.edu and sign in with their NetID. The enterprise plan provisions the right policy automatically — there's no extra step to opt out of training. 

# Install Claude Code on Windows (PowerShell)
winget install Anthropic.ClaudeCode
claude auth login
claude --version
