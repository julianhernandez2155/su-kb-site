# Claude — Frequently Asked Questions

This is a sample page for the Quartz + clementine styling integration spike. Content here is illustrative; the real Claude FAQ will land via the Confluence export pipeline in Stage 5 of the build.

## Data classification

> [!warning] University data with Claude
> Use the **`syr.edu`-signed-in** Claude account for any work involving university data. The personal Claude account is **not** approved for FERPA-protected information.

The [[index|Approved Tools list]] (page coming soon) is the canonical source for which AI tools are approved for which data classifications. Claude is approved for public and internal data; consult IT before using with confidential or FERPA-protected material.

## Retention and training

- **Chat retention**: 2 years by default on Claude.ai with SU's enterprise plan
- **Training opt-out**: SU's enterprise plan opts all university chats out of model training
- **Travel**: Claude can be used while traveling abroad subject to SU's international data-handling policies

## Premium seats

Premium Claude seats (with API access and Claude Code) cost approximately **$2,400/year** per seat. Available to IT staff and specific approved programs. Request via the [[claude-code-setup|Claude Code Setup]] flow (page coming soon).
