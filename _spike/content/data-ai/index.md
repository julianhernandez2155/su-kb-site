# Data & AI at Syracuse University

The **ITS Data & AI** department supports faculty, staff, and students using artificial intelligence tools at Syracuse University. This site replaces the prior Confluence space (`ITSAI`) as the public source for tool policies, setup guides, and example workflows.

| Clementine Platform | SU's own mentorAI platform — build a course-specific AI mentor (page coming soon) |
| AI - General Information | Approved tools list + cross-platform comparisons (page coming soon) |

## Why this site exists

The ITS Data & AI team needs students and staff to be able to:

1. **Get authoritative answers** about which AI tools are approved for which kinds of university data
2. **Find setup guides** for Claude Code, mentorAI, the M365 connector, and other surfaces
3. **Ask Claude itself** for the same answers via an org-wide Claude skill that routes the WebFetch tool to this site

> [!tip] Source of truth
> Pages on this site are exported from SU's Confluence by a Python pipeline. **Edit at the Confluence source**, not here — changes flow downstream on the next sync.
